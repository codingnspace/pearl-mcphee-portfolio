Reach Mobile

Work Project

Overview

Reach mobile is powerful tool app used by patients in the Conta Costa county health system to find local non profits and programs to enrich their lives and improve their health.

From a technical perspective, Reach Mobile is offered as a progressive web app and hybrid Android app. I worked as one of 2 JS developers to this project to beta.

  • Developed new React components to be used in both apps
  • Created and used REST API endpoints
  • Leveraged Redux and associated libraries to manage app data
  • Worked closely with Product and Design teams to establish product requirements
  • Led user focus groups at 2 California clinics
  • Ensured app used internationalization (i18n) to translate into 6 languages

Reach

Work Project

Overview

Reach is case management software that is currently used in over 200 major US hostpitals and clinics.

From a technical perspective, it was built on the front-end with React, Backbone, JQuery, LESS and HTML. On the backend: Apex and Salesforce were primarily used.

I was responsible for:

  • Converting legacy pages into modern React componets
  • Creating a pattern library for documentation
  • Maintained complexed data storage and flow in a secure and performant way
  • Writing unit test to increase test coverage of UI code
  • Updating the logic in backend Apex templates
